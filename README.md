##  验证介绍
安全宝网络验证针对的是平台级的多作者账户的验证(可用于运营或个人使用)，界面美观，功能强大
该验证的用户机制区别于所有验证（该验证可以绑定多个机器码或者IP地址）
用户前端：eleadmin
开发环境：PHP8.0 + MySQL5.7
